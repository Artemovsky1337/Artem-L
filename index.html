О красках

Краски — это смеси, используемые для покрытия поверхностей с целью их защиты, украшения или изменения их свойств. Они бывают различных цветов и составов, подходящих для разных целей.

Современные краски содержат пигменты, связующие вещества, растворители и добавки для улучшения их свойств.

Виды красок

  • Для Декора Подходит для: гостиной, кухни, холла
  • Для Интерьера Подходит для: гостиной, спальни, коридора
  • Для Фасада Подходит для: художественной живописи
